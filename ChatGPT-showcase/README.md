# Trolley Problem - ChatGPT Implementation

An interactive philosophy playground exploring **Consequentialism** and **Kantian Deontology** through the classic trolley problem scenario.

## Overview

This implementation demonstrates ChatGPT's approach to building an educational ethics application. It features a clean, minimalist design focused on clarity and accessibility.

## Features

- **Theory Explorer**: Interactive sections explaining Consequentialism and Deontology
- **Moral Dilemma Simulator**: The classic trolley problem with choice-based feedback
- **Decision Visualizer**: Side-by-side comparison of ethical reasoning approaches
- **Mini Ethics Quiz**: Test your understanding of philosophical concepts
- **Academic References**: Chicago-style citations for further reading

## Technical Approach

- **Simplicity First**: Single HTML file with embedded CSS and JavaScript
- **Tailwind CSS**: Utility-first styling for rapid, consistent design
- **Lightweight**: No build process required - just open in a browser
- **Responsive**: Mobile-friendly grid layouts
- **Interactive**: Simple JavaScript for user interactions

## Philosophy Concepts Covered

### Consequentialism
- Actions judged by outcomes
- Utilitarianism: maximize total happiness
- "The ends justify the means"

### Deontology (Kantian Ethics)
- Morality based on duties and rules
- Categorical Imperative
- Treat people as ends, not means

## Running Locally

Simply open `index.html` in any modern web browser:

```bash
open index.html
```

Or serve with a local server:

```bash
python -m http.server 8000
# Then visit http://localhost:8000
```

## Design Philosophy

This implementation prioritizes:
