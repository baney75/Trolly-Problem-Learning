# AI Agent Instructions for AI Ethics Experiments Showcase

Welcome to the AI Ethics Experiments Showcase repository! This project compares how different AI models (ChatGPT, Claude, and Gemini) implement the same interactive philosophy application.

## ⚠️ Important Rules

1. **DO NOT modify the AI-generated code**:
   - The directories `ChatGPT-showcase`, `Claude-showcase`, and `Gemini-showcase` contain the **unaltered outputs** from the respective AI models.
   - Do not attempt to fix bugs, refactor code, or update dependencies within these specific showcase directories unless explicitly instructed by the user, as the whole point of this repository is to preserve and study the original AI outputs.

2. **Dashboard Modifications**:
   - You may modify `index.html`, `README.md`, and any files at the root level to improve the showcase dashboard itself.
   - When modifying `index.html`, keep the styling consistent with the existing dark theme and typography (`Inter` and `JetBrains Mono`).

3. **Running the Showcase**:
   - The project is primarily static HTML/CSS/JS. You can use any static server (e.g., `python -m http.server 8000`) to test the root `index.html` file.
