# Gemini Implementation

A modern React-based interactive philosophy application exploring **Consequentialism** and **Kantian Deontology** through the Trolley Problem.

## Overview

This implementation demonstrates Gemini's approach to building a contemporary educational ethics application using modern web technologies and best practices.

- **Component-Based Architecture**: Modular React components for each section
- **Modern Stack**: React 18 + TypeScript + Vite
- **Type Safety**: Full TypeScript implementation

## Philosophy Covered

### Consequentialism
- Actions evaluated by outcomes
- Utilitarianism: maximize happiness, minimize suffering
- Key thinkers: Bentham, Mill, Singer

### Deontology
- Duty-based ethics from Immanuel Kant
- Categorical Imperative: universal laws
- Treat persons as ends, not means
- Key thinkers: Kant, Ross, Korsgaard
